# AgentCrew 品牌素材

## 使用说明

### Logo
- **文件**: `logo.svg`
- **用途**: 项目图标、GitHub 头像、Favicon
- **尺寸**: 200x200px（矢量，可任意缩放）

### 社交预览图
- **文件**: `social-preview.html`
- **用途**: GitHub 仓库的 Social Preview（1280×640px）

**生成 PNG 方法：**

1. **用浏览器打开** `social-preview.html`
2. **按 Command + Shift + 4**，然后按空格键，点击窗口截图
3. 或者使用 Chrome DevTools：
   - 按 F12 打开开发者工具
   - 按 Command + Shift + P
   - 输入 "Capture full size screenshot"
   - 保存为 `social-preview.png`

### 配色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 深蓝黑 | `#1a1a2e` | 主背景 |
| 深蓝 | `#16213e` | 次要背景 |
| 蓝色 | `#0f3460` | 点缀 |
| 珊瑚红 | `#e94560` | 强调色、Logo |

## 设计概念

- **中心节点** 代表协调者（Niko）
- **周围 6 个节点** 代表专业 Agent
- **连接线** 象征 A2A（Agent-to-Agent）通信网络
- **整体呈圆形** 寓意团队协作、循环迭代
